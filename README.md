# SCA Career Accelerator | Deployment README

## 🚀 Overview
The **SCA Career Accelerator** is a high-fidelity SaaS prototype designed to bridge the gap between technical expertise and strategic positioning. Using the **Strategic Context Analysis (SCA)** framework, it transforms raw career data into "Evolutionary Assets" (CVs and Cover Letters) optimized for high-stakes engineering roles and board-level review.

### Key Features:
- **Intake Flow:** Supports LinkedIn URLs, Text pasting, and File Uploads.
- **SCA Diagnostics:** Real-time alignment scoring and "Gap Heatmap" visualization.
- **Dual Output:** Generates matching Evolutionary CVs and Strategic Cover Letters.
- **Dark Mode UI:** Premium "Global Mastery" aesthetic with glassmorphism components.

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS (via CDN), Lucide Icons.
- **Visualization:** Chart.js for Radar-based Skill Balance Maps.
- **Logic:** Vanilla JavaScript (Prototyped for SCA Decision Logic).

## 🚢 Deployment (Vercel)

This project is optimized for **Vercel's zero-config deployment**.

### Option A: Via Vercel Dashboard
1. Go to [vercel.com](https://vercel.com) and sign in.
2. Click **"New Project"**.
3. Import this repository from your GitHub.
4. Vercel will automatically detect the `index.html`.
5. Click **"Deploy"**.

### Option B: Via Vercel CLI
If you have the Vercel CLI installed, run the following in your terminal:
```bash
vercel --prod
```
